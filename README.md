Программа - прототип для анализа настроения/слов/персонажей фильмов по сценам с визуализацией данных (пример входных данных - Scooby Doo_0267913).

Выполняет следующие задачи:

1. Получает базу данных фраз, которые произносят главные герои.
В базе обязательно содержатся номера сцен, говорящий и произносимый текст.
Для примера выдаётся строчка из сцены с самым большим числом реплик.

2. Выдаёт количество и список персонажей, которые произносят хотя бы одну реплику.
Происходит визуализация информации о количестве произносимых реплик в виде диаграмм.
Персонажи отсортированы по числу произносимых реплик (топ 20)

3. Выдаёт граф вразимодействий персонажей. Под взаимодействием подразумевается наличие произношение реплик в одной сцене.
Для каждой пары показывает количество сцен, в которых они взаимодействуют. Визуализирует самые взаимодействующие пары (топ 20)

4. Строит облако из 100 самых встречающихся слов среди реплик. Форма множественного числа учитывается вместе с формой единственного числа.
Слова типа: boy, boys, boy's, boys' считаются за одно слово. Из облако слов исключается 1000 самых употребляемых слов в английском языке (unigram_freq.csv)

5. Производится анализ настроения слов: считается доля позитивных (positive.txt), негативных (negative.txt),
общих (не позитивные и не негативные слова из 1000 самых употребимых) и нейтральных (все остальные слова) слов для каждого персонажа.
Также сделана визуализация настроения, слов, по сценам.
